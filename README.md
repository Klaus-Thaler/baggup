# BAGGUP

## This app offers data backup via your home Wi-Fi network. You can select individual folders and different types of files.

On Linux, a daemon can be used to query the app's web page with a Bash script and curl. This provides immediate security for your home network when the app is launched via your own Wi-Fi.
